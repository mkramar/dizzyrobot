## Overview
The root certificate catalog holds the entire trusted root certificate known to the SimpleLinkâ„¢ CC3120/CC3220 device. The catalog may change from time to time and needs to be programmed to the devices file system. If an older version already programmed to the devices file system, the program fails. the catalog holds only fingerprints of the original root CA's and the whole CA needs to be programmed to the file system when used.
